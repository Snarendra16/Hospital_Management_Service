# Hospital Management System

## Setup

1. **Backend Setup**
   ```powershell
   cd backend
   pip install -r requirements.txt
   ```

2. **Frontend Setup**
   ```powershell
   cd frontend
   npm install
   ```

## Running the Application


3. **Start Flask Backend** (In backend folder)
   ```powershell
   python app.py
   ```

4. **Start Vue Frontend** (In frontend folder)
   ```powershell
   npm run dev
   ```
